# Next.js rewrite (stable app router)

این پوشه نسخه بازنویسی‌شده پروژه با Next.js (App Router) است.

## Features
- ساختار پایدار Next.js با `app/`
- Metadata API برای SEO پویا
- OpenGraph/Twitter metadata
- Structured Data (JSON-LD) برای صفحه ویدیو
- استفاده از `next/image` برای بهینه‌سازی تصویر
- Dynamic import برای کاهش بار اولیه
- استخراج UI تکراری در `VideoListSection`

## Run
```bash
cd next
npm install
npm run dev
```
